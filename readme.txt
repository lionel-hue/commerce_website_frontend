1. Le Formulaire

Champs du Formulaire.

ID produit (ou clé primaire) : un identifiant unique pour chaque produit. Ceci ne mets pas sur le formulaire c'est dans la base de données on fera ça

Nom du produit : le nom du produit.

Description du produit : une brève description du produit.

Catégorie : la catégorie à laquelle appartient le produit (ex : fruits, produits laitiers, boissons).

Marque : la marque du produit.

Quantité en stock : le nombre d'unités disponibles.

Prix unitaire : le prix d'une unité du produit.

Date d'expiration (si applicable) : la date à laquelle le produit expire.<br>Code-barres : un code-barres ou un autre identifiant pour le produit.

Poids/Volume : la quantité de contenu en poids ou volume (ex : 500g, 1L).

Fournisseur : le nom ou l'identifiant du fournisseur du produit.

Date d'ajout : la date à laquelle le produit a été ajouté à la base de données.<br>Statut : pour indiquer si le produit est actif, en rupture de stock, ou discontinué.
